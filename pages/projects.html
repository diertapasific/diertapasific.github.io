Rupcara: AI Money Recognition for the Visually Impaired

  • Developed a deep learning-based tool to recognize Indonesian Rupiah banknotes using a fine-tuned ResNet-18 model.
  • Leveraged computer vision techniques for real-time banknote classification across multiple years (2011, 2016, 2022).
  • Implemented speech synthesis to provide instant, audio feedback of the detected banknote value.
  • Optimized image preprocessing and model inference for smooth performance on web platforms.
  • Created an interactive demo web app to showcase accessible AI technology in action.

Frezz: AI Fruit Freshness Detector

  • Uses deep learning to analyze fruit images and determine freshness levels.
  • Trained a Convolutional Neural Network (CNN) model using a dataset of fresh and non-fresh fruits.
  • Converted the pre-trained model to TensorFlow.js and integrated it into a Progressive Web Application (PWA).
  • Achieved an accuracy of 98% in classifying fresh and rotten apples, bananas, and oranges.

Telecom Customer Churn Analysis

  • Predicts customer churn using machine learning to help businesses retain customers.
  • Utilizes Exploratory Data Analysis (EDA) to uncover data patterns.
  • Trained a Random Forest model to predict churn based on customer behavior.
  • Performed feature importance analysis to identify key factors influencing churn.
  • Key insights: Total Day Charge, Total Day Minutes, and Customer Service Calls are top predictors of churn.
